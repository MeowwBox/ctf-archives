For transparency, TinyBall Deluxe publishes an archive of past draws. Older results are displayed as coloured squares for "accessibility reasons". This is not a security concern. Please do not break our lottery machine.

Note that remote has a PoW, so make sure you solve locally before trying remotely. There is a PoW solver provided in the template, you do not need to handle any logic, but keep it in mind when testing locally via Docker.
