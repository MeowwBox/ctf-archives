Sally’s Shell Shop has a problem: rare shells are disappearing from the return desk, but every record says the requests came through the normal customer portal. No staff passwords were stolen. No doors were forced. Just a few odd orders, strange profile pictures, and support tickets that seem to approve themselves. Find the path from customer to dock, figure out how the shop is being tricked, and recover Sally’s missing prize.
