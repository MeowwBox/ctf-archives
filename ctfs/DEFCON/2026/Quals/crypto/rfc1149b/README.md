Documented by f0xtr0t and bluepichu.

Alice, Bob, and Carol are being mean to me. They all got these cute trained pigeons and use them to send messages back and forth using some weird protocol they invented. But I want to join in the fun too, so I got a trained falcon who can intercept their messages so I can read and modify them! Unfortunately I'm running a little low on ink and whiteout, and if I leave to get more I might miss some juicy gossip, so I think I'm just going to have to make do with what I have. Can you help me send Bob a message about leaving me out? - Mallory
