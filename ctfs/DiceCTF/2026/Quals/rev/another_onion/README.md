Challenge instructions: you are given an example binary an_onion and an example input a_key. When you're ready, spin up an instance to download a real binary (without the key). You'll have 30 minutes to solve this real binary. Once you solve it, submit the token to the instance to get the flag.

You may attempt multiple times, but each instance will generate a different binary and a different solution token. All such generated binaries will be similar to an_onion, but rather than a 16 byte key, they will have 512 byte keys.
