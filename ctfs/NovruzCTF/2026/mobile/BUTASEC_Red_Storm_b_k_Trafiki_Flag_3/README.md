Artıq əlində “Secret Code” var. İndi serverlə əlaqə qurmağın vaxtıdır. Lakin tətbiq SSL Pinning ilə qorunur və adi proksi vasitəsilə trafiki görmək mümkün deyil. Şəbəkə trafikini “tut” və serverin göndərdiyi flag-i ələ keçir.

(Bu CTF-də flag-lər bir-biri ilə sıx bağlıdır. Bir mərhələni keçmədən digərinə keçmək texniki olaraq qeyri-mümkündür. Ona görə FLAG2 həll etdikdən sonra bunu həll etməyiniz məsləhətdir.)
