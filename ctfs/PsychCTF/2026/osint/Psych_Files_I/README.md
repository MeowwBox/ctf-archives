The Psych Files surfaced without explanation.

Forty-five clinical records attributed to the Hargrove Institute for Behavioral Health — dated, signed, processed, archived.

The documents are intact. The institution is not.

Names repeat. Addresses remain constant. Only the signage seems to have changed.

One entry does not conform to the rest of the archive. What is it?
