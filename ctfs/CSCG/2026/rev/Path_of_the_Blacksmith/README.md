Check out my new Minecraft server!

The server is running on version 1.21.10 of vanilla Minecraft and has the option online-mode set to false.

To connect to the server use the SSH command below to setup a port forward, after that just connect to the server at localhost:25565 in your Minecraft client.

After starting the instance, the server might not be fully responsive at first, just give it a minute and everything should work as intended.
