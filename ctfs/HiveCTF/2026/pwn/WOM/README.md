Signetics Model 25120 Fully-encoded, 9046 x N Random Access, Write Only Memory Emulator https://web.archive.org/web/20060317085650/http://www.national.com/rap/files/datasheet.pdf

This software module is suitable for inclusion into any project requiring absolute security and First In, Never Out (FINO) memory access. Also, this module is perfect for organizations with strict data retention policies.

Users can instantiate as many as 32 WOM instances to hold their important data. If they run out, user may set any full WOM free and rest assured that they met their regulatory retention requirements, and may reuse that WOM slot for new data [effectively, infinite storage!]
